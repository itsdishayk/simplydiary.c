### ‚ú® Key Features
* **Add Notes:** Save your daily thoughts instantly.
* **Persistent Storage:** Data is saved in `diary.txt`, so it's still there after you close the app.
* **Append Mode:** New entries are added to the end of the file, keeping your history safe.
* **Lightweight:** No heavy libraries, just pure C logic.
